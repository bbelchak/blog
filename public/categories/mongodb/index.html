automatic mongodb backups to s3

2011-01-20
#backups  #mongodb  #python 
One of the big problems with hosting your own database solution is that you have to do backups for it on a regular basis. Not only do you need to do backups for it, but you need to also keep backups offsite. Luckily, Amazon S3 allows a cheap and easy solution for your offsite backups. I found a shell script solution for handling MongoDB backups, but it only does local backups.
